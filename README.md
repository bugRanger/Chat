# Chat

Sample text and voice chat.

### Commands:
```sh
!connect -ip 127.0.0.1 -p 30010
!login -u user1
!send -u user2 -m Hello!
!call -u user2
!hangup -s 1234
!mute -r 1
!unmute -r 1
!logout
!disconnect
```
